환영합니다!

파이썬 튜토리얼에 오신 것을 환영합니다. 이번 시간에는 Google Colaboratory (Colab) 환경을 설정하고, 시간 흐름에 따른 데이터 분석을 위한 모든 준비를 마칠 것입니다.

📁

학습 자료 접근

모든 학습 자료는 Google Drive에 저장되어 있으며, 강의 자료(3-4강) 또는 웹사이트에서 다운로드 버튼을 통해 접근할 수 있습니다.

📝

Google Colab 시작

새로운 Colab 노트북을 생성하고, 파일 이름을 **'Introduction to Time Series Analysis'**로 변경하세요. 좌측 패널에서 **목차(Table of Contents)**, **파일(Files)** 섹션을 확인하세요.

☁️

Google Drive 마운트

Colab에서 Google Drive에 연결하려면, 좌측 파일 아이콘을 클릭하여 드라이브를 마운트하거나 다음 코드를 실행합니다:

from google.colab import drive
drive.mount('/content/drive')
마운트가 완료되면 'drive' 폴더가 보일 것입니다.

📂

작업 디렉토리 설정

데이터셋이 있는 특정 폴더로 이동합니다. Colab에서 디렉토리를 변경하려면 다음 명령어를 사용하고, 드라이브에서 해당 폴더의 **경로를 복사하여 붙여넣으세요**:

%cd [복사한 경로]
이제 CSV 데이터 파일에 쉽게 접근할 수 있습니다.

🚀

라이브러리 임포트 및 데이터 로드

필요한 라이브러리를 임포트하고, 비트코인 가격 데이터셋을 로드합니다. `df.head()`를 사용하여 데이터의 첫 5행을 미리 볼 수 있습니다.

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv('bitcoin_price.csv')
df.head()
데이터는 날짜, 시가, 고가, 저가, 종가, 조정 종가, 거래량으로 구성됩니다.
