3. 첫인상 파악: 데이터 시각화

데이터를 직접 눈으로 확인하는 것은 분석의 첫걸음입니다. 선 그래프를 통해 데이터의 전반적인 추세와 패턴을 파악할 수 있습니다.

 

4. 숨겨진 리듬 찾기: 계절성 분석

많은 시계열 데이터는 특정 주기를 가지고 반복되는 '계절성' 패턴을 보입니다. 이를 분해하고 이해하는 것은 예측 정확도를 높이는 핵심입니다.

시계열의 구성 요소

시계열 데이터는 주로 추세, 계절성, 그리고 불규칙한 나머지(잔차)로 구성됩니다.

 

월별 데이터 패턴

`month_plot`은 각 월별 데이터의 평균적인 경향을 보여주어 계절적 패턴을 쉽게 파악하도록 돕습니다.

 

5. 과거의 메아리: 자기상관 분석 (ACF)

자기상관 함수(ACF)는 현재의 데이터가 과거의 데이터와 얼마나 연관되어 있는지를 보여줍니다. 파란색 영역을 벗어나는 막대는 통계적으로 유의미한 상관관계를 의미합니다.

 

6. 직접적 영향 분석: 부분 자기상관 (PACF)

부분 자기상관 함수(PACF)는 다른 시점의 영향을 제외하고, 오직 두 시점 간의 순수한 상관관계를 보여줍니다. 예측 모델의 파라미터를 결정하는 데 중요한 역할을 합니다.

 

7. 예측 심판하기: 모델 성능 평가

좋은 예측 모델을 만들기 위해서는 모델의 성능을 객관적인 지표로 평가해야 합니다. RMSE, MAE, MAPE는 예측이 실제 값과 얼마나 차이 나는지를 나타내는 대표적인 지표입니다.

RMSE

오차의 크기에 민감

MAE

오차의 절대적인 평균

MAPE

오차를 백분율로 표시

8. 미래 내다보기: 최종 예측 시각화

분석의 최종 목표는 미래를 예측하는 것입니다. 과거 데이터(Train)를 기반으로 학습한 모델이 생성한 미래 예측(Forecast)을 시각적으로 확인하며 분석을 마무리합니다.
